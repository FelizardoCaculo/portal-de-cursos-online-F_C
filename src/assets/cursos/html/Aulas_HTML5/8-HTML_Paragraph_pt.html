8ª Aula - Parágrafos no HTML

O parágrafo no HTML o a tag ‹p› no HTML, é usada para definir um parágrafo na página web. Vamos pegar um exemplo simples para ver como ele funciona. É um ponto notável que o navegador web adiciona ele mesmo uma linha vazia antes e depois de um parágrafo. uma tag ‹p› do HTML indica o início de um novo parágrafo.


Nota: Se estiveres a usar várias tags ‹p› num ficheiro HTML, o navegador automaticamente adiciona uma linhaem branco entre os dois parágrafos.
