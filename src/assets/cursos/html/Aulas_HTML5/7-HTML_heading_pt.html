7ª Aula - Títulos no HTML

Um título no HTML ou tag ‹h› pode ser definido como título ou subtítuloque voce quer apresentar na página web. Quando voce escreve o texto dentro das tags ‹h1› à ‹h6›, ele é apresentado no navegador no formato negrito e o tamanho do texto depende do número do título.


Há 6 títulos diferentes no HTML que são definidos com o ‹h1› à ‹h6›, do mais alto nível ‹h1› (título principal) para o mais baixo nível ‹h6› (o título de menor importância).


‹h1› é a maior tag de título e o ‹h6› é a menor. ‹h1› é usada para o título mais importante e ‹h6› é usada para o menos importante.


Os títulos no HTML ajudam os motores de busca para entender e indexar a extrutura da página web.


Nota: A palavra-chave principal de todo conteúdo da página web deve ser apresentado pela tag de título ‹h1›.
