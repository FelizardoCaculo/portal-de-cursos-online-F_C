4ª Aula - Atributos HTML

  • Os atributos HTML são palavras especiais que providenciam informação adicional sobre os elementos, ou seja, os atributos são modificadores dos elementos HTML.
  • Cada elemento ou tag pode ter atributos, que definem o comportamento desse elemento.
  • Atributos devem sempre ser aplicados na tag de abertura.
  • O atributo deve sempre ser aplicado com o seu nome e valor par.
  • O nome e valor do atributo são case sensitive, e é recomendado pela W3C que deve sempre ser escrito somente com letras minúsculas.
  • Voce pode adicionar multiplos atributos num elemento HTML, mas precisa dar espaço entre os atributos.
