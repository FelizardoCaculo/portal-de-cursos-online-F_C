5ª Aula - Elementos HTML

Um ficheiro HTML é feito de elementos. Estes elementos são responsáveis por criar páginas web e define o conteúdo naquela página. Um elemento HTML usualmente consiste de uma tag de abertura ‹nome da tag›, tag de fechamento ‹/nome da tag› e o conteúdo inserido entre eles. Tecnicamente, um elemento é uma colecção de tag de abertura, atributos, tag de fechamento e o conteúdo entr eles.


Nota: Alguns elementos não tem tag de fechamento e conteúdo, estes elementos são chamados elementos vazios ou elementos de auto-fechamento.


Elementos vazios: Nem todos elementos HTML requerem ter uma tag de abertura e tag de fechamento, alguns elementos não tem conteúdo nem tag de fechamento, tais elementos são conhecidos como elementos vazios


Alguns elementos vazios são:
‹br› => Representa uma quebra de linha;
‹hr› => Representa uma linha horizontal; etc.


Elementos HTML Aninhados: Os elementos HTML podem ser aninhados, que significa um elemento pode conter outro elemento.


Elementos HTML em Bloco e em Linha


Por padrão de apresentação no ecrã e propósitos de estilização no HTML, os elementos são divididos em duas categorias:

  • Elementos em Bloco
  • Elementos em Linha

Elementos em Bloco:

  • Estes são os elemntos, que estruturam a parte principal da página web, dividindo a página em blocos coerentes Um abaixo do outro.
  • Um elemento em Bloco sempre começa com uma nova linhae toma para sí toda largura da página web, da esquerda para direita.
  • Estes elementos contem elementos em Bloco e elementos em Linha.
