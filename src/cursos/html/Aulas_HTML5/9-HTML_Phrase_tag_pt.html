9ª Aula - Tags de Frase no HTML

As tags frase no HTML são tags de propósito especial, que definem o significado estrutural de um bloco de texto ou semantica do texto. A seguir está a lista de tags frase, algumas das quais jão discutimos na Formatação do HTML.


  • Tag Abreviação: ‹abbr›
  • Tag Acronym: ‹acronym› (Já não é suportado no HTML5)
  • Tag Marcado: ‹mark›
  • Tag Strong: ‹strong›
  • Tag Enfatizado: ‹em›
  • Tag Definição: ‹dfn›
  • Tag Citação: ‹blockquote›
  • Tag Short quote: ‹q›
  • Tag Code: ‹code›
  • Tag teclado: ‹kbd›
  • Tag Endereço: ‹address›
