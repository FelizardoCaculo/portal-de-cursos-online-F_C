Introduction

To build websites, you have to know about HTML, it is a fundamental technology used to build the web page structure. HTML is used to specify if the content of the web must be recognized as a paragraph, a list, a title, a link, an image, a multimedia visualization, a form or one among many other elements avaliable or even a new element defined by you.

The major points of HTML are given below:


  • HTML, stands for HyperText Markup Language.
  • HTML, is used to create web pages and web applications.
  • HTML, is widely used language on the web.
  • We can create a static website by HTML only.
  • Technically, HTML is a Markup language rather than a programming language.
